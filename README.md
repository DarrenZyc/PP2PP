# PP2PP
CSSE432 2021S Computer Networks Project  
To run:    

    python node.py 1
     
 

To use:  

0. get the available commands

        usage
        
1. send direct message  

        message <id> -d <message>

    example:  

        message 3 -d hello


2. send flood message  
    
        message <id> -f <message>
        
    example:  

        message 3 -f flooding

3. save files

        save <file_path>

4. get file

        get <file_name>
    
5. delete file

        delete <file_name>
